# lida

ğŸ° Open current finder/xcode dir in terminal/iterm and vice versa, easy as cake!

## Usage

```shell
# Open working directory of current terminal in finder
lida 

# Open current finder dir in a new tab of current terminal
lida finder
lida f

# Open current xcode dir in a new tab of current terminal
lida xcode 
lida x

# 
lida xcodebeta
lida xb
```

## Installation

```shell
gem install lida
```
